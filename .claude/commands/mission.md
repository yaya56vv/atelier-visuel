# /mission — Agent Codeur (Phase 3 — Mission générique)

Tu es un agent codeur. Tu interviens sur une mission technique précise.

## AVANT TOUTE CHOSE — OBLIGATION DE LECTURE

Tu dois **d'abord et intégralement** lire **AGENT.md** (sections 0, 3, 4, 5, 6 en priorité).

Tu ne commences aucun travail tant que tu n'as pas :
1. Compris le sens du projet (section 2)
2. Pris connaissance de l'état réel du code (sections 4 et 5)
3. Intégré les règles de travail (sections 3 et 6)
4. Vérifié les zones interdites (section 4.3)

## TA MISSION

L'utilisateur va décrire la tâche après ce bloc d'introduction.

Avant de coder, assure-toi d'avoir compris :
- Quel est l'objectif réel (pas juste "ajouter X", mais pourquoi)
- Ce que tu as le droit de modifier et ce que tu dois préserver
- Si doute → question courte

## PROTOCOLE

### Avant
1. Relis ta mission
2. Vérifie les zones interdites/fragiles (§4.2, §4.3)
3. Mini-plan (5 lignes, dans la discussion)

### Pendant
- Modifie petit, valide souvent
- Supprime l'ancien code après validation du nouveau
- Logs temporaires uniquement pour debug

### Après
- Teste ce que tu as fait
- Supprime ancien code et logs temporaires
- Liste les fichiers modifiés/supprimés

## FIN DE MISSION — OBLIGATOIRE

### 1) Mise à jour d'AGENT.md
Sections §4, §5, §7, §8 selon ce qui a changé.
Aucun autre fichier de documentation autorisé.

### 2) Compte-rendu dans le chat
- Ce que j'ai fait (3-5 points)
- Ce qui a changé dans le code
- Ce que j'ai mis à jour dans AGENT.md
- Comment vérifier que ça marche
- Risques ou points d'attention
- Ce qui reste à faire (si incomplet)
